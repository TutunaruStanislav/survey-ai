Необходимо сгенерировать полнофункциональное веб-приложение мини-анкета, используя лучшие практики разработки и чистую архитектуру.
Требуемый функционал:
Backend (Python) + простой сервер
- 2 API endpoints:
- GET /questions — возвращает список из 5 жестко заданных вопросов анкеты
- POST /answers — принимает ответы пользователя и сохраняет их в мапу
Frontend (HTML + JavaScript + CSS)
- Загружает вопросы с backend и динамически рендерит форму с вопросами
- Отправляет заполненные ответы на POST /answers и показывает сообщение "Спасибо!" после успешной отправки
